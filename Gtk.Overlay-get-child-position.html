Gtk.Overlay::get-child-position

function callback(overlay, widget:Gtk.Widget, ):Boolean;

The ::get-child-position signal is emitted to determine the position and size of any overlay child widgets. A handler for this signal should fill allocation with the desired position and size for widget, relative to the 'main' child of @overlay.

The default handler for this signal uses the widget's halign and valign properties to determine the position and gives the widget its natural size (except that an alignment of Gtk.Align.fill will cause the overlay to be full-width/height). If the main child is a Gtk.ScrolledWindow, the overlays are placed relative to its contents.

Return: true if the allocation has been filled

overlay

instance of Gtk.Overlay that is emitting the signal

widget

the child widget to position
