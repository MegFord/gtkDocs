Gtk.EntryCompletion:cell-area

"cell-area"             Gtk.CellArea                : Read / Write / Construct Only

The Gtk.CellArea used to layout cell renderers in the treeview column.

If no area is specified when creating the entry completion with Gtk.EntryCompletion.new_with_area a horizontally oriented Gtk.CellAreaBox will be used.

Since 3.0
