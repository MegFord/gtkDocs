Gtk.Widget.prototype.register_window

function register_window(window:Gdk.Window):void {
    // Gjs wrapper for gtk_widget_register_window()
}

Registers a Gdk.Window with the widget and sets it up so that the widget recieves events for it. Call Gtk.unregister_window when destroying the window.

Before 3.8 you needed to call Gdk.set_user_data directly to set this up. This is now deprecated and you should use Gtk.Widget.prototype.register_window instead. Old code will keep working as is, although some new features like transparency might not work perfectly.

Since 3.8

window

a Gdk.Window
