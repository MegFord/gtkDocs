test_find_label

function test_find_label(widget:Gtk.Widget, label_pattern:String):Gtk.Widget {
    // Gjs wrapper for gtk_test_find_label()
}

This function will search widget and all its descendants for a GtkLabel widget with a text string matching label_pattern. The label_pattern may contain asterisks '*' and question marks '?' as placeholders, g_pattern_match() is used for the matching. Note that locales other than "C" tend to alter (translate" label strings, so this function is genrally only useful in test programs with predetermined locales, see Gtk.test_init for more details.

Since 2.14

widget

Valid label or container widget.

label_pattern

Shell-glob pattern to match a label string.

Returns

a GtkLabel widget if any is found.
