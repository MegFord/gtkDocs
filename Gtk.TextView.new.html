Gtk.TextView.new

function new():Gtk.Widget {
    // Gjs wrapper for gtk_text_view_new()
}

Creates a new Gtk.TextView. If you don't call Gtk.TextView.prototype.set_buffer before using the text view, an empty default buffer will be created for you. Get the buffer with Gtk.TextView.prototype.get_buffer. If you want to specify your own buffer, consider Gtk.new_with_buffer.

Returns

a new Gtk.TextView
