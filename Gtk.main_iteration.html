main_iteration

function main_iteration():Boolean {
    // Gjs wrapper for gtk_main_iteration()
}

Runs a single iteration of the mainloop.

If no events are waiting to be processed GTK+ will block until the next event is noticed. If you don't want to block look at main_iteration_do or check if any events are pending with events_pending first.

Returns

true if main_quit has been called for the innermost mainloop
