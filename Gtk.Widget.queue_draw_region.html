Gtk.Widget.prototype.queue_draw_region

function queue_draw_region(region:cairo.Region):void {
    // Gjs wrapper for gtk_widget_queue_draw_region()
}

Invalidates the area of widget defined by region by calling Gdk.invalidate_region on the widget's window and all its child windows. Once the main loop becomes idle (after the current batch of events has been processed, roughly), the window will receive expose events for the union of all regions that have been invalidated.

Normally you would only use this function in widget implementations. You might also use it to schedule a redraw of a Gtk.DrawingArea or some portion thereof.
