Gtk.IconFactory.prototype.add_default

function add_default():void {
    // Gjs wrapper for gtk_icon_factory_add_default()
}

Adds an icon factory to the list of icon factories searched by Gtk.lookup_icon_set. This means that, for example, Gtk.new_from_stock will be able to find icons in factory. There will normally be an icon factory added for each library or application that comes with icons. The default icon factories can be overridden by themes.
