Gtk.CssProvider.prototype.load_from_file

function load_from_file(file:Gio.File):Boolean {
    // Gjs wrapper for gtk_css_provider_load_from_file()
}

Loads the data contained in file into css_provider, making it clear any previously loaded information.

file

Gio.File pointing to a file to load

Returns

true. The return value is deprecated and false will only be returned for backwards compatibility reasons if an @error is not null and a loading error occured. To track errors while loading CSS, connect to the GtkCssProvider::parsing-error signal.
