Gtk.PrintOperation:current-page

"current-page"             Number                : Read / Write

The current page in the document.

If this is set before Gtk.PrintOperation.prototype.run, the user will be able to select to print only the current page.

Note that this only makes sense for pre-paginated documents.

Since 2.10
