Gtk.Assistant.prototype.set_current_page

function set_current_page(page_num:Number):void {
    // Gjs wrapper for gtk_assistant_set_current_page()
}

Switches the page to page_num.

Note that this will only be necessary in custom buttons, as the assistant flow can be set with Gtk.set_forward_page_func.

Since 2.10

page_num

index of the page to switch to, starting from 0. If negative, the last page will be used. If greater than the number of pages in the assistant, nothing will be done.
