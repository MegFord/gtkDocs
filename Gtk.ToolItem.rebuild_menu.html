Gtk.ToolItem.prototype.rebuild_menu

function rebuild_menu():void {
    // Gjs wrapper for gtk_tool_item_rebuild_menu()
}

Calling this function signals to the toolbar that the overflow menu item for tool_item has changed. If the overflow menu is visible when this function it called, the menu will be rebuilt.

The function must be called when the tool item changes what it will do in response to the Gtk.create-menu-proxy signal.

Since 2.6
