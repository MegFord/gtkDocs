Gtk.Action:icon-name

"icon-name"             String                : Read / Write

The name of the icon from the icon theme.

Note that the stock icon is preferred, if the Gtk.stock-id property holds the id of an existing stock icon, and the Gio.Icon is preferred if the Gtk.Action:gicon property is set.

This is an appearance property and thus only applies if Gtk.use-action-appearance is true.

Since 2.10
