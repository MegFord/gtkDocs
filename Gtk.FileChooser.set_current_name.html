Gtk.FileChooser.prototype.set_current_name

function set_current_name(name:filename):void {
    // Gjs wrapper for gtk_file_chooser_set_current_name()
}

Sets the current name in the file selector, as if entered by the user. Note that the name passed in here is a UTF-8 string rather than a filename. This function is meant for such uses as a suggested name in a "Save As..." dialog. You can pass "Untitled.doc" or a similarly suitable suggestion for the name.

If you want to preselect a particular existing file, you should use Gtk.set_filename or Gtk.set_uri instead. Please see the documentation for those functions for an example of using Gtk.FileChooser.prototype.set_current_name as well.

Since 2.4

name

the filename to use, as a UTF-8 string
