Gtk.PlacesSidebar::open-location

function callback(places_sidebar, location:GObject.Object, open_flags:Gtk.PlacesOpenFlags, ):void;

The places sidebar emits this signal when the user selects a location in it. The calling application should display the contents of that location; for example, a file manager should show a list of files in the specified location.

Since 3.10

places_sidebar

instance of Gtk.PlacesSidebar that is emitting the signal

location

Gio.File to which the caller should switch.

open_flags

a single value from Gtk.PlacesOpenFlags specifying how the location should be opened.
