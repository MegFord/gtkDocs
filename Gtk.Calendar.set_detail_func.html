Gtk.Calendar.prototype.set_detail_func

function set_detail_func(func:Gtk.CalendarDetailFunc):void {
    // Gjs wrapper for gtk_calendar_set_detail_func()
}

Installs a function which provides Pango markup with detail information for each day. Examples for such details are holidays or appointments. That information is shown below each day when Gtk.show-details is set. A tooltip containing with full detail information is provided, if the entire text should not fit into the details area, or if Gtk.show-details is not set.

The size of the details area can be restricted by setting the Gtk.detail-width-chars and Gtk.detail-height-rows properties.

Since 2.14

func

a function providing details for each day.
