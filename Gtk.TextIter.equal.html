Gtk.TextIter.prototype.equal

function equal(rhs:Gtk.TextIter):Boolean {
    // Gjs wrapper for gtk_text_iter_equal()

Tests whether two iterators are equal, using the fastest possible mechanism. This function is very fast; you can expect it to perform better than e.g. getting the character offset for each iterator and comparing the offsets yourself. Also, it's a bit faster than Gtk.TextIter.prototype.compare.
