Gtk.HSeparator

const Gtk = imports.gi.Gtk;

let h_separator = new Gtk.HSeparator();

The Gtk.HSeparator widget is a horizontal separator, used to group the widgets within a window. It displays a horizontal line with a shadow to make it appear sunken into the interface.

<note> The Gtk.HSeparator widget is not used as a separator within menus. To create a separator in a menu create an empty Gtk.SeparatorMenuItem widget using Gtk.new and add it to the menu with Gtk.append. </note>

GtkHSeparator has been deprecated, use Gtk.Separator instead.
