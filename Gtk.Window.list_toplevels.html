list_toplevels

function list_toplevels():[Gtk.Widget] {
    // Gjs wrapper for gtk_window_list_toplevels()
}

Returns a list of all existing toplevel windows. The widgets in the list are not individually referenced. If you want to iterate through the list and perform actions involving callbacks that might destroy the widgets, you <emphasis>must</emphasis> call <literal>g_list_foreach (result, (GFunc)g_object_ref, NULL)</literal> first, and then unref all the widgets afterwards.

Returns

list of toplevel widgets
