events_pending

function events_pending():Boolean {
    // Gjs wrapper for gtk_events_pending()
}

Checks if any events are pending.

This can be used to update the UI and invoke timeouts etc. while doing some time intensive computation.

<example> <title>Updating the UI during a long computation</title> <programlisting> /&ast; computation going on... &ast;/

while (gtk_events_pending ()) gtk_main_iteration ();

/&ast; ...computation continued &ast;/ </programlisting> </example>

Returns

true if any events are pending, false otherwise
