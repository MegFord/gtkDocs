main_iteration_do

function main_iteration_do(blocking:Boolean):Boolean {
    // Gjs wrapper for gtk_main_iteration_do()
}

Runs a single iteration of the mainloop. If no events are available either return or block depending on the value of blocking.

blocking

true if you want GTK+ to block if no events are pending

Returns

true if main_quit has been called for the innermost mainloop
