Gtk.RecentManager.prototype.lookup_item

function lookup_item(uri:String):Gtk.RecentInfo {
    // Gjs wrapper for gtk_recent_manager_lookup_item()
}

Searches for a URI inside the recently used resources list, and returns a structure containing informations about the resource like its MIME type, or its display name.

Since 2.10

uri

a URI

Returns

a Gtk.RecentInfo structure containing information about the resource pointed by uri, or null if the URI was not registered in the recently used resources list. Free with Gtk.RecentInfo.prototype.unref.
