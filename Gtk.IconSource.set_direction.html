Gtk.IconSource.prototype.set_direction

function set_direction(direction:Gtk.TextDirection):void {
    // Gjs wrapper for gtk_icon_source_set_direction()
}

Sets the text direction this icon source is intended to be used with.

Setting the text direction on an icon source makes no difference if the text direction is wildcarded. Therefore, you should usually call Gtk.set_direction_wildcarded to un-wildcard it in addition to calling this function.

direction

text direction this source applies to
