Gtk.Container.prototype.set_focus_hadjustment

function set_focus_hadjustment(adjustment:Gtk.Adjustment):void {
    // Gjs wrapper for gtk_container_set_focus_hadjustment()
}

Hooks up an adjustment to focus handling in a container, so when a child of the container is focused, the adjustment is scrolled to show that widget. This function sets the horizontal alignment. See Gtk.get_hadjustment for a typical way of obtaining the adjustment and Gtk.set_focus_vadjustment for setting the vertical adjustment.

The adjustments have to be in pixel units and in the same coordinate system as the allocation for immediate children of the container.

adjustment

an adjustment which should be adjusted when the focus is moved among the descendents of container
