Gtk.HeaderBar.prototype.set_custom_title

function set_custom_title(title_widget:Gtk.Widget):void {
    // Gjs wrapper for gtk_header_bar_set_custom_title()
}

Sets a custom title for the Gtk.HeaderBar. The title should help a user identify the current view. This supercedes any title set by Gtk.set_title or Gtk.set_subtitle. You should set the custom title to null, for the header title label to be visible again.

Since 3.10

title_widget

a custom widget to use for a title
