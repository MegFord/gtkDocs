Gtk.Adjustment.prototype.clamp_page

function clamp_page(lower:Number, upper:Number):void {
    // Gjs wrapper for gtk_adjustment_clamp_page()
}

Updates the Gtk.value property to ensure that the range between lower and upper is in the current page (i.e. between Gtk.value and Gtk.value + #GtkAdjustment:page_size). If the range is larger than the page size, then only the start of it will be in the current page. A Gtk.changed signal will be emitted if the value is changed.

lower

the lower value.

upper

the upper value.
