Gtk.IconSource.prototype.set_direction_wildcarded

function set_direction_wildcarded(setting:Boolean):void {
    // Gjs wrapper for gtk_icon_source_set_direction_wildcarded()
}

If the text direction is wildcarded, this source can be used as the base image for an icon in any Gtk.TextDirection. If the text direction is not wildcarded, then the text direction the icon source applies to should be set with Gtk.IconSource.prototype.set_direction, and the icon source will only be used with that text direction.

Gtk.IconSet prefers non-wildcarded sources (exact matches) over wildcarded sources, and will use an exact match when possible.

setting

true to wildcard the text direction
