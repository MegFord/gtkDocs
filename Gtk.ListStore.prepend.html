Gtk.ListStore.prototype.prepend

function prepend():void {
    // Gjs wrapper for gtk_list_store_prepend()
}

Prepends a new row to list_store. iter will be changed to point to this new row. The row will be empty after this function is called. To fill in values, you need to call Gtk.set or Gtk.set_value.
