Gtk.CellRendererCombo:text-column

"text-column"             Number                : Read / Write

Specifies the model column which holds the possible values for the combo box.

Note that this refers to the model specified in the model property, <emphasis>not</emphasis> the model backing the tree view to which this cell renderer is attached.

Gtk.CellRendererCombo automatically adds a text cell renderer for this column to its combo box.

Since 2.6
