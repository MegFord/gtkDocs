Gtk.Calendar

const Gtk = imports.gi.Gtk;

let calendar = new Gtk.Calendar({
    day: value,
    detail_height_rows: value,
    detail_width_chars: value,
    month: value,
    no_month_change: value,
    show_day_names: value,
    show_details: value,
    show_heading: value,
    show_week_numbers: value,
    year: value,
});

Gtk.Calendar is a widget that displays a Gregorian calendar, one month at a time. It can be created with Gtk.new.

The month and year currently displayed can be altered with Gtk.select_month. The exact day can be selected from the displayed month using Gtk.select_day.

To place a visual marker on a particular day, use Gtk.mark_day and to remove the marker, Gtk.unmark_day. Alternative, all marks can be cleared with Gtk.clear_marks.

The way in which the calendar itself is displayed can be altered using Gtk.set_display_options.

The selected date can be retrieved from a Gtk.Calendar using Gtk.get_date.

Users should be aware that, although the Gregorian calendar is the legal calendar in most countries, it was adopted progressively between 1582 and 1929. Display before these dates is likely to be historically incorrect.
