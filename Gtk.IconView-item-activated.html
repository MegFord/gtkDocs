Gtk.IconView::item-activated

function callback(icon_view, path:Gtk.TreePath, ):void;

The ::item-activated signal is emitted when the method Gtk.IconView.prototype.item_activated is called, when the user double clicks an item with the "activate-on-single-click" property set to false, or when the user single clicks an item when the "activate-on-single-click" property set to true. It is also emitted when a non-editable item is selected and one of the keys: Space, Return or Enter is pressed.

icon_view

instance of Gtk.IconView that is emitting the signal

path

the Gtk.TreePath for the activated item
