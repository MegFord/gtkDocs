Gtk.PrintOperationPreview::got-page-size

function callback(print_operation_preview, context:Gtk.PrintContext, page_setup:Gtk.PageSetup, ):void;

The ::got-page-size signal is emitted once for each page that gets rendered to the preview.

A handler for this signal should update the context according to page_setup and set up a suitable cairo context, using Gtk.PrintContext.prototype.set_cairo_context.

print_operation_preview

instance of Gtk.PrintOperationPreview that is emitting the signal

context

the current Gtk.PrintContext

page_setup

the Gtk.PageSetup for the current page
