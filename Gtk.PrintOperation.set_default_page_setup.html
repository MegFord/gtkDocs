Gtk.PrintOperation.prototype.set_default_page_setup

function set_default_page_setup(default_page_setup:Gtk.PageSetup):void {
    // Gjs wrapper for gtk_print_operation_set_default_page_setup()
}

Makes default_page_setup the default page setup for op.

This page setup will be used by Gtk.PrintOperation.prototype.run, but it can be overridden on a per-page basis by connecting to the Gtk.request-page-setup signal.

Since 2.10

default_page_setup

a Gtk.PageSetup, or null
