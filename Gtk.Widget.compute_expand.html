Gtk.Widget.prototype.compute_expand

function compute_expand(orientation:Gtk.Orientation):Boolean {
    // Gjs wrapper for gtk_widget_compute_expand()
}

Computes whether a container should give this widget extra space when possible. Containers should check this, rather than looking at Gtk.get_hexpand or Gtk.get_vexpand.

This function already checks whether the widget is visible, so visibility does not need to be checked separately. Non-visible widgets are not expanded.

The computed expand value uses either the expand setting explicitly set on the widget itself, or, if none has been explicitly set, the widget may expand if some of its children do.
