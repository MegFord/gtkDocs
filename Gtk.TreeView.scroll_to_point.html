Gtk.TreeView.prototype.scroll_to_point

function scroll_to_point(tree_x:Number, tree_y:Number):void {
    // Gjs wrapper for gtk_tree_view_scroll_to_point()
}

Scrolls the tree view such that the top-left corner of the visible area is tree_x, tree_y, where tree_x and tree_y are specified in tree coordinates. The tree_view must be realized before this function is called. If it isn't, you probably want to be using Gtk.TreeView.prototype.scroll_to_cell.

If either tree_x or tree_y are -1, then that direction isn't scrolled.

tree_x

X coordinate of new top-left pixel of visible area, or -1

tree_y

Y coordinate of new top-left pixel of visible area, or -1
