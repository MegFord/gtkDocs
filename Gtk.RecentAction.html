Gtk.RecentAction

const Gtk = imports.gi.Gtk;

let recent_action = new Gtk.RecentAction({
    show_numbers: value,
});

A Gtk.RecentAction represents a list of recently used files, which can be shown by widgets such as Gtk.RecentChooserDialog or Gtk.RecentChooserMenu.

To construct a submenu showing recently used files, use a Gtk.RecentAction as the action for a &lt;menuitem&gt;. To construct a menu toolbutton showing the recently used files in the popup menu, use a Gtk.RecentAction as the action for a &lt;toolitem&gt; element.
