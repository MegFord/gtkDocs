Gtk.Tooltip.prototype.set_icon_from_gicon

function set_icon_from_gicon(gicon:Gio.Icon, size:Number):void {
    // Gjs wrapper for gtk_tooltip_set_icon_from_gicon()
}

Sets the icon of the tooltip (which is in front of the text) to be the icon indicated by gicon with the size indicated by size. If gicon is null, the image will be hidden.

Since 2.20

gicon

a Gio.Icon representing the icon, or null

size

a stock icon size
