Gtk.TreeSelection.prototype.get_selected_rows

function get_selected_rows():[Gtk.TreePath] {
    // Gjs wrapper for gtk_tree_selection_get_selected_rows()
}

Creates a list of path of all selected rows. Additionally, if you are planning on modifying the model after calling this function, you may want to convert the returned list into a list of Gtk.TreeRowReference<!-- -->s. To do this, you can use Gtk.TreeRowReference.new.

To free the return value, use: |[ g_list_free_full (list, (GDestroyNotify) gtk_tree_path_free); ]|

Since 2.2

Returns

A GLib.List containing a Gtk.TreePath for each selected row.
