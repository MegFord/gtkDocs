Gtk.PrintOperation::update-custom-widget

function callback(print_operation, widget:Gtk.Widget, setup:Gtk.PageSetup, settings:Gtk.PrintSettings, ):void;

Emitted after change of selected printer. The actual page setup and print settings are passed to the custom widget, which can actualize itself according to this change.

Since 2.18

print_operation

instance of Gtk.PrintOperation that is emitting the signal

widget

the custom widget added in create-custom-widget

setup

actual page setup

settings

actual print settings
