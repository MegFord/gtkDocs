Gtk.Label:max-width-chars

"max-width-chars"             Number                : Read / Write

The desired maximum width of the label, in characters. If this property is set to -1, the width will be calculated automatically.

See the section on <link linkend="label-text-layout">text layout</link> for details of how Gtk.width-chars and Gtk.Label:max-width-chars determine the width of ellipsized and wrapped labels.

Since 2.6
