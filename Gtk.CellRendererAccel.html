Gtk.CellRendererAccel

const Gtk = imports.gi.Gtk;

let cell_renderer_accel = new Gtk.CellRendererAccel({
    accel_key: value,
    accel_mode: value,
    accel_mods: value,
    keycode: value,
});

Gtk.CellRendererAccel displays a keyboard accelerator (i.e. a key combination like <keycombo><keycap>Control</keycap><keycap>a</keycap></keycombo>. If the cell renderer is editable, the accelerator can be changed by simply typing the new combination.

The Gtk.CellRendererAccel cell renderer was added in GTK+ 2.10.
