Gtk.Editable::delete-text

function callback(editable, start_pos:Number, end_pos:Number, ):void;

This signal is emitted when text is deleted from the widget by the user. The default handler for this signal will normally be responsible for deleting the text, so by connecting to this signal and then stopping the signal with g_signal_stop_emission(), it is possible to modify the range of deleted text, or prevent it from being deleted entirely. The start_pos and end_pos parameters are interpreted as for Gtk.Editable.prototype.delete_text.

editable

instance of Gtk.Editable that is emitting the signal

start_pos

the starting position

end_pos

the end position
