Gtk.IconInfo.prototype.get_filename

function get_filename():filename {
    // Gjs wrapper for gtk_icon_info_get_filename()
}

Gets the filename for the icon. If the Gtk.IconLookupFlags.use_builtin flag was passed to Gtk.lookup_icon, there may be no filename if a builtin icon is returned; in this case, you should use Gtk.IconInfo.prototype.get_builtin_pixbuf.

Since 2.4

Returns

the filename for the icon, or null if Gtk.IconInfo.prototype.get_builtin_pixbuf should be used instead. The return value is owned by GTK+ and should not be modified or freed.
