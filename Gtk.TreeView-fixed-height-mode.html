Gtk.TreeView:fixed-height-mode

"fixed-height-mode"             Boolean                : Read / Write

Setting the ::fixed-height-mode property to true speeds up Gtk.TreeView by assuming that all rows have the same height. Only enable this option if all rows are the same height. Please see Gtk.TreeView.prototype.set_fixed_height_mode for more information on this option.

Since 2.4
