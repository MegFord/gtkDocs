Gtk.PlacesSidebar.prototype.get_nth_bookmark

function get_nth_bookmark(n:Number):Gio.File {
    // Gjs wrapper for gtk_places_sidebar_get_nth_bookmark()
}

This function queries the bookmarks added by the user to the places sidebar, and returns one of them. This function is used by Gtk.FileChooser to implement the "Alt-1", "Alt-2", etc. shortcuts, which activate the cooresponding bookmark.

Since 3.10

n

index of the bookmark to query

Returns

The bookmark specified by the index n, or #NULL if no such index exist. Note that the indices start at 0, even though the file chooser starts them with the keyboard shortcut "Alt-1".
