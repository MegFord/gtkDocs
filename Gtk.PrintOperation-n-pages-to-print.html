Gtk.PrintOperation:n-pages-to-print

"n-pages-to-print"             Number                : Read

The number of pages that will be printed.

Note that this value is set during print preparation phase (Gtk.PrintStatus.preparing), so this value should never be get before the data generation phase (Gtk.PrintStatus.generating_data). You can connect to the Gtk.PrintOperation::status-changed signal and call Gtk.PrintOperation.prototype.get_n_pages_to_print when print status is Gtk.PrintStatus.generating_data. This is typically used to track the progress of print operation.

Since 2.18
