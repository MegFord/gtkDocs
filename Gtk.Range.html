Gtk.Range

const Gtk = imports.gi.Gtk;

let range = new Gtk.Range({
    adjustment: value,
    fill_level: value,
    inverted: value,
    lower_stepper_sensitivity: value,
    restrict_to_fill_level: value,
    round_digits: value,
    show_fill_level: value,
    upper_stepper_sensitivity: value,
});

Gtk.Range is the common base class for widgets which visualize an adjustment, e.g Gtk.Scale or Gtk.Scrollbar.

Apart from signals for monitoring the parameters of the adjustment, Gtk.Range provides properties and methods for influencing the sensitivity of the "steppers". It also provides properties and methods for setting a "fill level" on range widgets. See Gtk.set_fill_level.
