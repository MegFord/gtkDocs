Gtk.Widget.prototype.get_valign

function get_valign():Gtk.Align {
    // Gjs wrapper for gtk_widget_get_valign()
}

Gets the value of the Gtk.valign property.

For backwards compatibility reasons this method will never return Gtk.Align.baseline, but instead it will convert it to Gtk.Align.fill. If your widget want to support baseline aligned children it must use Gtk.get_valign_with_baseline.

Returns

the vertical alignment of widget, ignoring baseline alignment
