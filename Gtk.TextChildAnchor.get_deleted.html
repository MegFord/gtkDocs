Gtk.TextChildAnchor.prototype.get_deleted

function get_deleted():Boolean {
    // Gjs wrapper for gtk_text_child_anchor_get_deleted()
}

Determines whether a child anchor has been deleted from the buffer. Keep in mind that the child anchor will be unreferenced when removed from the buffer, so you need to hold your own reference (with g_object_ref()) if you plan to use this function &mdash; otherwise all deleted child anchors will also be finalized.

Returns

true if the child anchor has been deleted from its buffer
