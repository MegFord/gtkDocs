Gtk.Builder.new

function new():Gtk.Builder {
    // Gjs wrapper for gtk_builder_new()
}

Creates a new empty builder object.

This function is only useful if you intend to make multiple calls to Gtk.Builder.prototype.add_from_file, Gtk.Builder.prototype.add_from_resource or Gtk.Builder.prototype.add_from_string in order to merge multiple UI descriptions into a single builder.

Most users will probably want to use Gtk.new_from_file, Gtk.new_from_resource or Gtk.new_from_string.

Since 2.12

Returns

a new (empty) Gtk.Builder object
