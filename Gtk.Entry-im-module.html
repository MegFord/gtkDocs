Gtk.Entry:im-module

"im-module"             String                : Read / Write

Which IM (input method) module should be used for this entry. See Gtk.IMContext.

Setting this to a non-null value overrides the system-wide IM module setting. See the GtkSettings Gtk.gtk-im-module property.
