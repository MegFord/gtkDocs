Gtk.Widget.prototype.grab_focus

function grab_focus():void {
    // Gjs wrapper for gtk_widget_grab_focus()
}

Causes widget to have the keyboard focus for the Gtk.Window it's inside. widget must be a focusable widget, such as a Gtk.Entry; something like Gtk.Frame won't work.

More precisely, it must have the %GTK_CAN_FOCUS flag set. Use Gtk.set_can_focus to modify that flag.

The widget also needs to be realized and mapped. This is indicated by the related signals. Grabbing the focus immediately after creating the widget will likely fail and cause critical warnings.
