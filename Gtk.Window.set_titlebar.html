Gtk.Window.prototype.set_titlebar

function set_titlebar(titlebar:Gtk.Widget):void {
    // Gjs wrapper for gtk_window_set_titlebar()
}

Sets a custom titlebar for window.

If you set a custom titlebar, GTK+ will do its best to convince the window manager not to put its own titlebar on the window. Depending on the system, this function may not work for a window that is already visible, so you set the titlebar before calling Gtk.Widget.prototype.show.

Since 3.10

titlebar

the widget to use as titlebar
