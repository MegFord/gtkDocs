Gtk.CellRendererToggle

const Gtk = imports.gi.Gtk;

let cell_renderer_toggle = new Gtk.CellRendererToggle({
    activatable: value,
    active: value,
    inconsistent: value,
    indicator_size: value,
    radio: value,
});

Gtk.CellRendererToggle renders a toggle button in a cell. The button is drawn as a radio or a checkbutton, depending on the Gtk.radio property. When activated, it emits the Gtk.toggled signal.
