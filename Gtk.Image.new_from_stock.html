Gtk.Image.new_from_stock

function new_from_stock(stock_id:String, size:Number):Gtk.Widget {
    // Gjs wrapper for gtk_image_new_from_stock()
}

Creates a Gtk.Image displaying a stock icon. Sample stock icon names are #GTK_STOCK_OPEN, #GTK_STOCK_QUIT. Sample stock sizes are #GTK_ICON_SIZE_MENU, #GTK_ICON_SIZE_SMALL_TOOLBAR. If the stock icon name isn't known, the image will be empty. You can register your own stock icon names, see Gtk.IconFactory.prototype.add_default and Gtk.IconFactory.prototype.add.

stock_id

a stock icon name

size

a stock icon size

Returns

a new Gtk.Image displaying the stock icon
