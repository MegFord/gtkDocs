Gtk.AppChooserWidget:default-text

"default-text"             String                : Read / Write

The Gtk.AppChooserWidget:default-text property determines the text that appears in the widget when there are no applications for the given content type. See also Gtk.AppChooserWidget.prototype.set_default_text.
