Gtk.Entry.prototype.set_cursor_hadjustment

function set_cursor_hadjustment(adjustment:Gtk.Adjustment):void {
    // Gjs wrapper for gtk_entry_set_cursor_hadjustment()
}

Hooks up an adjustment to the cursor position in an entry, so that when the cursor is moved, the adjustment is scrolled to show that position. See Gtk.get_hadjustment for a typical way of obtaining the adjustment.

The adjustment has to be in pixel units and in the same coordinate system as the entry.

Since 2.12

adjustment

an adjustment which should be adjusted when the cursor is moved, or null
