Gtk.Button.prototype.set_image

function set_image(image:Gtk.Widget):void {
    // Gjs wrapper for gtk_button_set_image()
}

Set the image of button to the given widget. The image will be displayed if the label text is null or if Gtk.always-show-image is true. You don't have to call Gtk.show on image yourself.

Since 2.6

image

a widget to set as the image for the button
